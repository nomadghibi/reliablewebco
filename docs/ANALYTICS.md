# Analytics & Tracking

## Tools
- GA4 (required)
- Optional: Meta Pixel, TikTok Pixel (add-on)

## Events (name -> trigger)
- cta_primary_click -> clicks "Start My $499 Page"
- cta_call_click -> clicks "Book a 10-Min Call"
- pricing_view -> pricing section in viewport
- package_select -> clicks on a package button
- form_start -> user focuses first form input
- form_submit -> form submit success
- outbound_stripe_click -> clicks stripe payment link
- outbound_calendar_click -> clicks calendly link

## Conversions
- form_submit
- outbound_calendar_click
- outbound_stripe_click
