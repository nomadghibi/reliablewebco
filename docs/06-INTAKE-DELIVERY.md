# Intake + Delivery (Speed Protected)

## Intake Requirements (client must provide)
- Business name + phone + email
- Service list + service area
- 3 competitor links they respect
- Logo (or initial text-only)
- Any photos (optional)

## 24-Hour Landing Page Timeline
Clock starts when intake is complete + payment received.

## Revision Policy
- Landing page: 1 revision round
- Website-in-a-Week: 1 revision round
Additional rounds billed hourly.

## Acceptance
Project is complete when:
- Page is live on provided domain (or temporary URL)
- Forms tested
- Mobile + desktop checked
