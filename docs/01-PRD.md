# Product Requirements Document (PRD)

# PRD — ReliableWebStudio.com (Marketing Site)

## One-liner
"Florida businesses online fast — landing pages in 24 hours, full websites in a week."

## Primary Objective
Convert local service businesses into paid projects quickly.

## Target Customers
- HVAC, plumbing, cleaning, dental, landscaping, home services
- Small businesses needing leads now (not “branding exercises”)

## Core Offers
1) 24-Hour Landing Page — $499
2) Website-in-a-Week (5 pages) — $2,000
3) Growth Website (8–12 pages) — $3,500–$4,500
4) Web App / Portal (login, roles, DB) — Starts $8,000+

## Conversion Events
- Primary: “Book a Call”
- Secondary: “Buy Landing Page ($499)”
- Lead capture: contact form + SMS/call button

## Pages (v1)
- Home (long-form landing)
- Pricing
- Portfolio (can start with 2–4 samples / placeholders)
- Process
- FAQ
- Contact

## Trust Signals
- Clear scope + delivery times
- Guarantee (simple + confident)
- Testimonials (even if initial are “pilot clients”)
- “What you get” checklists

## Tech (v1)
- Next.js + TypeScript + Tailwind
- Static deploy on Vercel
- Simple form handling (email service later)
