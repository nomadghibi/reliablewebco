# Outreach Templates (Florida Local Web Design)

## Usage Rules
- Keep messages short and specific.
- Personalize first line using their business name and one real observation.
- Use one CTA only: book call or request audit.
- Follow-up at least 3 times before closing the loop.

## Template 1: Cold Email (Free Website Analysis)
Subject: Quick website analysis for {{BusinessName}}

Hi {{FirstName}},

I reviewed {{BusinessName}} and noticed a few conversion/SEO issues that are likely costing leads.

I can send you a free analysis with:
- top priority fixes (P0/P1/P2)
- quick wins for calls/forms
- local SEO improvements for {{City}}

If you want it, reply with your main website URL and I will send it next business day.

- {{YourName}}
Reliable Web Studio

## Template 2: Follow-up #1 (48 hours)

Worth sending?

## Objection Responses
- "Too expensive":
  This is built to produce leads fast, not just look better. We can start with a smaller scope and scale.
- "Not ready yet":
  No problem. I can send a phased plan so you know exactly what to do when timing is right.
- "Need to think":
  Makes sense. What part is unclear: scope, timing, or budget? I can simplify it in one page.

## Weekly Outreach Targets
- New outreach: 50 contacts/week
- Follow-ups: 75 touches/week
- Calls booked: 6+/week
- Proposal sends: 3+/week
