# Design System

## Visual Style
Modern, clean, high-trust, local-service SaaS vibe.
Goal: look premium but simple. Strong CTA buttons.

## Layout
- Max width: 1100–1200px
- Generous spacing, clear headings, readable line length
- Reusable section blocks

## Typography
- Headings: bold, large, minimal words
- Body: simple, readable, not dense
- Button text: action-based

## Color Guidelines
- Primary: strong CTA color (choose 1, keep consistent)
- Neutral background: white / light gray
- Accent: subtle for badges and highlights
- Use contrast compliant colors

## Components
- Navbar with sticky CTA
- Pricing cards (highlight “Most Popular”)
- Feature list with icons
- Timeline steps
- FAQ accordion
- Testimonials cards
- Portfolio/demo cards
- CTA strip component

## UX Rules
- CTA appears in hero, mid-page, and footer
- Keep forms short
- Use guarantees + scope boundaries near pricing
